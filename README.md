# Построение минимального остовного дерева

Визуальное приложение для построения графов и нахождения минимальных остовных деревьев с помощью алгоритмов Прима и Краскала.

## Возможности

- **Визуальное построение графов**
- **Два алгоритма MST**
- **Интерактивное редактирование**
- **Импорт/Экспорт**
- **Визуализация результатов**

## Установка и запуск

### Требования
- Python 3.7+
- PySide6

### Установка зависимостей
```
pip install PySide6
```

### Запуск приложения
```
python main.py
```

## Использование

### Режимы работы

1. **Выбрать**
2. **Добавить ноду**
3. **Добавить ребро**
4. **Удалить**

### Алгоритмы

- **Алгоритм Прима**
- **Алгоритм Краскала**

### Сохранение и загрузка

- **Экспорт**
- **Импорт**

## Тестирование

Для запуска тестов:
```
pytest test_graph_app.py -v
```

Тесты покрывают:
- Создание и удаление вершин и рёбер
- Корректность алгоритмов Прима и Краскала
- Экспорт и импорт графов
- Обработку граничных случаев

## Формат .json файла

Графы сохраняются в JSON формате:
```json
{
  "points": [
    {"index": 0, "x": 100.0, "y": 200.0},
    {"index": 1, "x": 300.0, "y": 150.0}
  ],
  "edges": [
    {"source_index": 0, "dest_index": 1, "weight": 1.5}
  ]
}
```
